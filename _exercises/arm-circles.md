# Arm Circles: Shoulder Warm-Up

> **At a Glance:** Simple circular movements that warm up and mobilize the shoulder joints.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Shoulder joints |
| **Best For** | Warm-up, shoulder mobility |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 1-2 minutes |
| **Frequency** | Daily as warm-up |

---

## How to Perform

1. Stand with arms out to sides
2. Make small circles forward
3. Gradually increase size
4. Reverse direction
5. 10-15 circles each direction

---

## Variations

- Small circles
- Large circles
- Single arm
- With light weights

---

## Benefits

- Warms up shoulders
- Improves circulation
- Prepares for exercise
- Quick and easy
