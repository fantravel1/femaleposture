# Warrior I: Standing Strength Pose

> **At a Glance:** A powerful standing pose that strengthens legs and opens the chest.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Legs, hips, core, shoulders |
| **Best For** | Leg strength, hip opening |
| **Difficulty** | Intermediate |
| **Equipment** | None |
| **Time** | 2-3 minutes per side |
| **Frequency** | Daily |

---

## How to Perform

1. Step one foot back (3-4 feet)
2. Back foot angled 45 degrees
3. Square hips forward
4. Bend front knee over ankle
5. Reach arms overhead
6. Hold 30-60 seconds

---

## Benefits

- Builds leg strength
- Stretches hip flexors
- Opens chest and shoulders
- Improves balance
