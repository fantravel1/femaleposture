# Mountain Climbers: Dynamic Core

> **At a Glance:** A challenging exercise that combines core work with cardiovascular conditioning.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Core, shoulders, hip flexors |
| **Best For** | Core strength, cardio |
| **Difficulty** | Intermediate |
| **Equipment** | None |
| **Time** | 5 minutes |
| **Frequency** | 3 times per week |

---

## How to Perform

1. Start in high plank position
2. Drive one knee toward chest
3. Quickly switch legs
4. Keep hips level (don't bounce up)
5. Core stays engaged
6. Maintain steady rhythm

### Speed Variations

- Slow: Focus on form
- Moderate: Steady pace
- Fast: Cardio emphasis

---

## Common Mistakes

- Hips bouncing up and down
- Hands creeping forward
- Not bringing knees high enough
- Holding breath
