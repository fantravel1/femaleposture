# Frog Stretch: Inner Thigh Opener

> **At a Glance:** An intense stretch for tight inner thighs and hips, common in yoga and mobility work.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Adductors, hip flexors |
| **Best For** | Inner thigh flexibility, hip mobility |
| **Difficulty** | Intermediate |
| **Equipment** | Mat, padding for knees |
| **Time** | 3-5 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Start on hands and knees
2. Widen knees out to sides
3. Turn feet outward, inner edges on floor
4. Lower to forearms if comfortable
5. Shift hips back toward heels
6. Hold 1-3 minutes

### Safety Tips

- Move slowly into position
- Stop if sharp pain
- Use padding under knees
- Start with mild stretch

---

## Benefits

- Deep inner thigh stretch
- Improves hip mobility
- Opens groin area
- Counteracts sitting
