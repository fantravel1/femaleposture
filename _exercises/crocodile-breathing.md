# Crocodile Breathing: Core Reset

> **At a Glance:** Lying face-down while breathing teaches proper 360-degree rib expansion.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Diaphragm, core, rib cage |
| **Best For** | Breathing mechanics, core reset |
| **Difficulty** | Beginner |
| **Equipment** | Mat |
| **Time** | 5 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Lie face down (prone)
2. Stack hands, rest forehead on hands
3. Breathe in deeply through nose
4. Feel belly push into floor
5. Feel ribs expand sideways
6. Exhale slowly
7. Repeat for 2-3 minutes

### Why Prone Position

- Floor provides feedback
- Encourages belly expansion
- Limits chest breathing
- Naturally relaxing

---

## Benefits

- Teaches proper breathing
- Activates deep core
- Reduces stress
- Great for core reset
