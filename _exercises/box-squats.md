# Box Squats: Safe Squatting Technique

> **At a Glance:** Box squats provide a target depth and build confidence for proper squatting mechanics.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Quadriceps, glutes, hamstrings |
| **Best For** | Squat technique, confidence building |
| **Difficulty** | Beginner |
| **Equipment** | Sturdy box, bench, or chair |
| **Time** | 5-10 minutes |
| **Frequency** | 3 times per week |

---

## How to Perform

1. Stand with box/bench behind you
2. Feet shoulder-width apart
3. Squat down until seated on box
4. Pause briefly (don't collapse)
5. Drive through heels to stand

### Form Tips

- Control the descent—don't drop
- Keep knees tracking over toes
- Chest stays up throughout
- Pause, don't bounce

---

## Benefits

- Teaches proper depth
- Builds confidence in squatting
- Reduces fear of falling
- Allows practice of hip hinge
- Good for beginners

---

## Progressions

1. High box → lower box
2. Bodyweight → add weight
3. Long pause → quick touch
4. Remove box when ready
