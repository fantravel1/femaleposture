# Thoracic Rotation: Upper Back Mobility

> **At a Glance:** Rotation exercises that target the often-stiff thoracic spine.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Thoracic spine |
| **Best For** | Upper back mobility, posture |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 5 minutes |
| **Frequency** | Daily |

---

## Seated Version

1. Sit tall, feet flat
2. Cross arms over chest
3. Rotate torso one direction
4. Keep hips facing forward
5. Hold 3-5 seconds
6. Rotate other direction
7. Repeat 10 times each side

---

## Quadruped Version

1. Hands and knees position
2. One hand behind head
3. Rotate elbow toward opposite elbow
4. Then rotate up toward ceiling
5. Follow elbow with eyes
6. Repeat 10 times, switch sides
