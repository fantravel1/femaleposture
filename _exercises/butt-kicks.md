# Butt Kicks: Hamstring Warm-Up

> **At a Glance:** A dynamic warm-up that prepares hamstrings for activity.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Hamstrings, quads (stretch) |
| **Best For** | Warm-up, running prep |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 1-2 minutes |
| **Frequency** | As warm-up |

---

## How to Perform

1. Stand tall or move forward
2. Kick heels up toward glutes
3. Alternate quickly
4. Keep thighs relatively stationary
5. Arms pump naturally

---

## Benefits

- Warms up hamstrings
- Prepares for running
- Dynamic quad stretch
- Improves leg turnover
