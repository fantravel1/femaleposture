# Standing Marches: Hip Flexor Activation

> **At a Glance:** A simple movement that activates hip flexors and challenges balance.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Hip flexors, core |
| **Best For** | Warm-up, balance |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 2-3 minutes |
| **Frequency** | Daily as warm-up |

---

## How to Perform

1. Stand tall
2. Lift one knee to hip height
3. Hold 1-2 seconds
4. Lower with control
5. Alternate legs
6. Keep torso stable
7. March for 20-30 reps

---

## Benefits

- Warms up hip flexors
- Improves balance
- Core activation
- Low impact
