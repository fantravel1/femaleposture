# Diaphragmatic Breathing: Core Foundation

> **At a Glance:** Proper breathing is the foundation of core function and posture, activating deep stabilizing muscles.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Diaphragm, transverse abdominis, pelvic floor |
| **Best For** | Core activation, stress relief, posture |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 5-10 minutes |
| **Frequency** | Multiple times daily |

---

## How to Perform

1. Lie on back, knees bent
2. Place one hand on chest, one on belly
3. Breathe in through nose
4. Let belly rise (not chest)
5. Exhale slowly through mouth
6. Belly falls naturally
7. Repeat 10-20 breaths

### Progression

- Add in sitting position
- Practice while standing
- Incorporate during daily activities

---

## Benefits

- Activates deep core
- Reduces stress and tension
- Improves rib cage mobility
- Foundation for all exercises
