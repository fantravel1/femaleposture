# Foam Roll Upper Back: Thoracic Release

> **At a Glance:** Rolling the upper back releases tension from sitting and improves extension.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Thoracic spine, upper back muscles |
| **Best For** | Upper back mobility, tension relief |
| **Difficulty** | Beginner |
| **Equipment** | Foam roller |
| **Time** | 3-5 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Roller across upper back
2. Support head with hands
3. Hips off floor
4. Roll upper back area
5. Pause on tight spots
6. Can extend over roller for mobility

---

## Tips

- Stay on upper back only
- Don't roll neck or lower back
- Use moderate pressure
- Breathe and relax
