# Upward Dog: Full Back Extension

> **At a Glance:** A stronger back extension where hips lift off the floor.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Back extensors, chest, shoulders |
| **Best For** | Back strength, chest opening |
| **Difficulty** | Intermediate |
| **Equipment** | Mat |
| **Time** | 3 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Start face down
2. Hands next to ribs
3. Press up, lifting chest AND thighs off floor
4. Only hands and tops of feet touch floor
5. Roll shoulders back
6. Hold 15-30 seconds

### Difference from Cobra

- Cobra: Hips stay on floor
- Up dog: Hips and thighs lift
