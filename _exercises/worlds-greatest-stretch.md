# World's Greatest Stretch: Full Body Mobility

> **At a Glance:** A comprehensive mobility movement that addresses multiple areas.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Hips, thoracic spine, hamstrings |
| **Best For** | Full body mobility, warm-up |
| **Difficulty** | Intermediate |
| **Equipment** | None |
| **Time** | 5 minutes |
| **Frequency** | Daily as warm-up |

---

## How to Perform

1. Start in push-up position
2. Step right foot outside right hand
3. Drop left knee (optional)
4. Rotate right arm up toward ceiling
5. Follow hand with gaze
6. Bring hand back down
7. Straighten front leg, hamstring stretch
8. Return to start, switch sides

---

## Benefits

- Opens hip flexors
- Rotates thoracic spine
- Stretches hamstrings
- Engages multiple planes
