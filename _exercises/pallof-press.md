# Pallof Press: Anti-Rotation Core

> **At a Glance:** Pressing against rotational resistance trains the core to resist twisting forces.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Obliques, transverse abdominis, core |
| **Best For** | Core stability, anti-rotation |
| **Difficulty** | Intermediate |
| **Equipment** | Cable machine or resistance band |
| **Time** | 5 minutes |
| **Frequency** | 3 times per week |

---

## How to Perform

1. Stand sideways to cable/band anchor
2. Hold handle at chest level
3. Feet shoulder-width apart
4. Press handle straight out
5. Resist rotationâ€”core stays square
6. Hold 2-3 seconds
7. Return to chest
8. Complete reps, then switch sides

---

## Variations

### Half-Kneeling
- One knee down
- Greater stability challenge

### Overhead Pallof
- Press up instead of out
- Different challenge angle

## Benefits

- Functional core strength

- Real-world stability
