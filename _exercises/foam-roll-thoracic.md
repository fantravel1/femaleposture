# Thoracic Foam Rolling: Upper Back Release

> **At a Glance:** Using a foam roller on the upper back releases tension and improves extension mobility.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Thoracic spine, upper back muscles |
| **Best For** | Upper back mobility, tension relief |
| **Difficulty** | Beginner |
| **Equipment** | Foam roller |
| **Time** | 3-5 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Lie on foam roller (positioned across upper back)
2. Support head with hands
3. Lift hips off floor
4. Roll slowly up and down upper back
5. Pause on tight spots
6. Can extend back over roller
7. Avoid lower back and neck

---

## Benefits

- Releases upper back tension
- Improves thoracic extension
- Counteracts slouching
- Feels immediately relieving

---

## Safety

- Avoid rolling lower back
- Don't roll over neck
- Use moderate pressure
- Stop if sharp pain
