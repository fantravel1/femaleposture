# Push-Ups: Upper Body Strength

> **At a Glance:** The classic exercise that builds upper body and core strength.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Chest, shoulders, triceps, core |
| **Best For** | Upper body strength |
| **Difficulty** | Beginner to Advanced |
| **Equipment** | None |
| **Time** | 5-10 minutes |
| **Frequency** | 2-3 times per week |

---

## How to Perform

1. Start in high plank position
2. Hands slightly wider than shoulders
3. Core engaged, body in straight line
4. Lower chest toward floor
5. Elbows at 45-degree angle
6. Push back up
7. Repeat

---

## Progressions

1. **Wall push-ups** - Easiest
2. **Incline push-ups** - Hands on bench
3. **Knee push-ups** - Knees on floor
4. **Full push-ups** - Standard
5. **Decline push-ups** - Feet elevated
