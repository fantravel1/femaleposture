# Heel Slides: Gentle Core Activation

> **At a Glance:** A low-load core exercise perfect for beginners or rehabilitation.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Transverse abdominis, core |
| **Best For** | Core activation, rehabilitation |
| **Difficulty** | Beginner |
| **Equipment** | Mat |
| **Time** | 5 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Lie on back, knees bent
2. Maintain neutral spine
3. Engage core gently
4. Slowly slide one heel out
5. Keep pelvis stableâ€”no movement
6. Slide heel back
7. Alternate legs
8. Repeat 10 times each

---

## Focus

- Pelvis stays completely still
- Lower back maintains position
- Core works to stabilize
