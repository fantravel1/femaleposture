# Crescent Lunge: Standing Hip Flexor Stretch

> **At a Glance:** A standing lunge that stretches hip flexors while building leg strength.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Hip flexors, quads, glutes |
| **Best For** | Hip mobility, leg strength |
| **Difficulty** | Intermediate |
| **Equipment** | None |
| **Time** | 2-3 minutes per side |
| **Frequency** | Daily |

---

## How to Perform

1. Start in lunge position
2. Back knee hovers off floor
3. Front knee over ankle
4. Reach arms overhead
5. Sink hips low
6. Hold 30-60 seconds

---

## Benefits

- Stretches hip flexors
- Builds leg strength
- Challenges balance
- Opens chest
