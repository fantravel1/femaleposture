# Forearm Stretch: Wrist & Forearm Release

> **At a Glance:** Essential stretches for forearms strained by keyboard and mouse use.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Forearm flexors and extensors |
| **Best For** | Wrist pain, typing strain |
| **Difficulty** | Beginner |
| **Equipment** | None |
| **Time** | 2-3 minutes |
| **Frequency** | Multiple times daily |

---

## Flexor Stretch

1. Extend arm in front, palm up
2. Use other hand to pull fingers down
3. Feel stretch in inner forearm
4. Hold 20-30 seconds

## Extensor Stretch

1. Extend arm in front, palm down
2. Use other hand to pull fingers down
3. Feel stretch on top of forearm
4. Hold 20-30 seconds
