# Lat Foam Rolling: Side Body Release

> **At a Glance:** Rolling the lats releases tension that can restrict shoulder mobility and affect posture.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Latissimus dorsi |
| **Best For** | Shoulder mobility, overhead reach |
| **Difficulty** | Beginner |
| **Equipment** | Foam roller |
| **Time** | 3 minutes |
| **Frequency** | Daily |

---

## How to Perform

1. Lie on side with roller under armpit area
2. Bottom arm extended overhead
3. Roll from armpit to mid-ribs
4. Pause on tender spots
5. Rotate slightly to target different fibers
6. Switch sides

---

## Benefits

- Releases lat tightness
- Improves overhead reach
- Helps with rounded shoulders
- Better breathing mechanics
