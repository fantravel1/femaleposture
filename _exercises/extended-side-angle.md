# Extended Side Angle: Full Side Stretch

> **At a Glance:** A deep side stretch from fingertips to back heel.

---

## Quick Facts

| | |
|---|---|
| **Targets** | Side body, hips, shoulders |
| **Best For** | Side flexibility, hip opening |
| **Difficulty** | Intermediate |
| **Equipment** | None |
| **Time** | 2-3 minutes per side |
| **Frequency** | Daily |

---

## How to Perform

1. From Warrior II
2. Lower front forearm to front thigh
3. Or hand to floor inside foot
4. Top arm reaches overhead by ear
5. Create long line from back heel to fingertips
6. Hold 30-60 seconds
