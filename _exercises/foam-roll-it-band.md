# IT Band Foam Rolling: Lateral Leg Release

> **At a Glance:** Rolling the outer thigh can help release IT band tension, though strengthening is also key.

---

## Quick Facts

| | |
|---|---|
| **Targets** | IT band, TFL, vastus lateralis |
| **Best For** | IT band syndrome, knee pain |
| **Difficulty** | Intermediate (can be intense) |
| **Equipment** | Foam roller |
| **Time** | 3-5 minutes per leg |
| **Frequency** | As needed |

---

## How to Perform

1. Lie on side, roller under outer thigh
2. Support yourself with arms
3. Roll from hip to just above knee
4. Pause on tender spots (30-60 seconds)
5. Cross top leg over for less pressure
6. Switch sides

### Note

- This area can be very sensitive
- Lighter pressure is often better
- Hip strengthening is equally important
