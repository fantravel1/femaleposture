# Diastasis Recti et Posture

Le diastasis recti—la séparation des muscles grands droits de l'abdomen pendant la grossesse—affecte jusqu'à deux tiers des femmes pendant la grossesse et peut persister longtemps après l'accouchement. Cette séparation impacte significativement la posture, la fonction du core et la vie quotidienne.

La bonne nouvelle : avec la bonne compréhension et des exercices ciblés, la plupart des femmes peuvent améliorer significativement ou guérir complètement leur diastasis.

## Qu'Est-Ce Que le Diastasis Recti?

### L'Anatomie

Vos muscles grands droits de l'abdomen courent verticalement le long de votre abdomen, reliés au centre par une bande de tissu conjonctif appelée ligne blanche. Pendant la grossesse, les hormones ramollissent ce tissu tandis que l'utérus en croissance pousse vers l'extérieur, causant la séparation des muscles.

**Séparation normale pendant la grossesse :** Jusqu'à 2 cm est considéré physiologique
**Diagnostic de diastasis :** Séparation supérieure à 2-2,5 cm ou plus de 2 largeurs de doigt

### Pourquoi C'est Important pour la Posture

Votre core est un cylindre :
- **Haut :** Diaphragme
- **Bas :** Plancher pelvien
- **Côtés :** Obliques et transverse abdominal
- **Avant :** Grand droit de l'abdomen

Quand la paroi frontale est compromise :
- Le soutien spinal diminue
- D'autres muscles compensent
- La posture souffre
- Le mal de dos se développe souvent

## Auto-Évaluation

### Comment Vérifier le Diastasis

1. Allongez-vous sur le dos, genoux pliés, pieds à plat
2. Placez les doigts sur la ligne médiane, au-dessus du nombril
3. Soulevez légèrement la tête et les épaules
4. Sentez la séparation entre les muscles
5. Mesurez combien de doigts entrent dans l'espace
6. Vérifiez aussi sous le nombril

## Exercices Sûrs

### Phase 1 : Connexion (Semaines 1-6 post-partum)

**Respiration diaphragmatique :**
1. Allongez-vous sur le dos ou sur le côté
2. Main sur le ventre
3. Inspirez : le ventre s'élargit doucement
4. Expirez : le ventre se retire doucement
5. 5-10 minutes par jour

### Phase 2 : Réhabilitation (Semaines 6-12)

**Ponts de fessiers :**
1. Allongez-vous sur le dos, genoux pliés
2. Expirez et soulevez les hanches
3. Maintenez 3 secondes
4. Abaissez avec contrôle
5. 10-15 répétitions

### Phase 3 : Progression (3+ mois)

- Dead bug modifié
- Bird dog
- Planche modifiée (inclinée)
- Exercices fonctionnels progressifs

## Exercices à ÉVITER

- Abdominaux traditionnels (crunches)
- Sit-ups
- Planche complète (initialement)
- Exercices qui causent un renflement
- Soulèvement lourd sans technique

## Points Clés

1. **Le diastasis est très commun**—affecte jusqu'à 2/3 des femmes enceintes
2. **Il affecte directement la posture**—le core est la base
3. **Le traitement est spécifique**—pas n'importe quel exercice de core fonctionne
4. **Certains exercices aggravent**—évitez les mauvais
5. **La plupart s'améliorent significativement**—avec la bonne approche
6. **La patience est nécessaire**—la récupération prend du temps
