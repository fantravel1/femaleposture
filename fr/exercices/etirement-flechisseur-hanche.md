# Étirement des Fléchisseurs de Hanche

Les fléchisseurs de hanche se raccourcissent quand nous sommes assis longtemps. Cet étirement soulage la tension et aide à restaurer la longueur musculaire.

## Données Rapides

| | |
|---|---|
| **Cible** | Psoas, iliaque, droit fémoral |
| **Idéal Pour** | Tension de position assise, bascule pelvienne, douleur lombaire |
| **Difficulté** | Débutant |
| **Équipement** | Tapis (optionnel) |
| **Temps** | 2-3 minutes |

## Pourquoi Ça Fonctionne

- Allonge les muscles qui se raccourcissent en position assise
- Aide à corriger la bascule pelvienne antérieure
- Réduit la tension dans le bas du dos
- Améliore l'extension de hanche

## Étirement Genou au Sol

### Instructions

1. Agenouillez-vous avec un genou au sol
2. L'autre pied devant, genou à 90°
3. Gardez le torse droit
4. Rentrez légèrement le coccyx
5. Déplacez le poids vers l'avant
6. Sentez l'étirement devant la hanche arrière
7. Tenez 30-45 secondes
8. Changez de côté

### Intensifier

- Levez le bras du côté que vous étirez
- Penchez-vous légèrement du côté opposé
- Gardez le core actif

## Étirement Debout

### Instructions

1. Faites un grand pas en avant
2. Genou avant plié
3. Jambe arrière tendue
4. Rentrez le coccyx
5. Poussez les hanches vers l'avant
6. Tenez 30 secondes

## Étirement sur le Canapé

### Instructions

1. Agenouillez-vous devant un canapé
2. Placez un pied sur le canapé (dessus du pied sur le coussin)
3. L'autre jambe devant, pied au sol
4. Gardez le torse droit
5. Étirement plus intense

## Erreurs Courantes

- **Cambrer le dos** — Rentrez le coccyx, activez le core
- **Se pencher en avant** — Gardez le torse droit
- **Forcer trop** — Progressez graduellement
- **Oublier de respirer** — Respirez profondément

## Quand Étirer

- Après de longues périodes assise
- Comme partie de l'échauffement
- Après l'exercice
- Avant de dormir

## Points Clés

1. **La constance est clé** — Faites-le régulièrement
2. **Rentrez le coccyx** — Intensifie l'étirement correctement
3. **Core actif** — Protège le bas du dos
4. **Les deux côtés** — Étirez les deux hanches
5. **Pas de rebond** — Gardez l'étirement stable
