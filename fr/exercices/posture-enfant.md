# Posture de l'Enfant: Repos et Étirement

La posture de l'enfant est une position de repos qui étire doucement le dos, les hanches et les épaules. Elle est parfaite pour les pauses, la relaxation et la récupération.

## Données Rapides

| | |
|---|---|
| **Cible** | Bas du dos, hanches, épaules, cou |
| **Idéal Pour** | Repos, étirement doux, relaxation |
| **Difficulté** | Débutant |
| **Équipement** | Tapis (optionnel) |
| **Temps** | 1-3 minutes |

## Pourquoi Ça Fonctionne

- Allonge doucement la colonne
- Détend les muscles du dos
- Étire les hanches et les cuisses
- Position de repos pour le système nerveux
- Facilite la respiration profonde

## Instructions

### Position de Base

1. Agenouillez-vous sur le sol
2. Asseyez-vous sur les talons
3. Écartez les genoux largeur des hanches (ou plus)
4. Penchez-vous vers l'avant
5. Étendez les bras devant ou le long du corps
6. Appuyez le front sur le sol
7. Détendez tout le corps
8. Respirez profondément

### Durée

- Maintenez 30 secondes à plusieurs minutes
- Respirez profondément
- Détendez-vous plus à chaque expiration

## Variations

### Bras Étendus

- Bras devant, paumes vers le bas
- Plus d'étirement des épaules et du dos

### Bras le Long du Corps

- Bras le long du corps, paumes vers le haut
- Plus relaxant

### Genoux Ensemble

- Genoux ensemble
- Étire plus le bas du dos

### Genoux Écartés

- Genoux plus écartés
- Plus d'espace pour le torse
- Idéal pendant la grossesse

## Erreurs Courantes

- **Forcer la position** — Doit être confortable
- **Retenir la respiration** — Respirez profondément
- **Tendre les épaules** — Laissez-les détendues
- **Ne pas utiliser de support si nécessaire** — Utilisez des coussins

## Quand Utiliser

- Entre les exercices comme repos
- À la fin de la pratique
- Pendant les pauses travail
- Avant de dormir
- Quand vous avez besoin d'un moment de calme

## Points Clés

1. **C'est du repos, pas du travail** — Détendez-vous complètement
2. **Respirez profondément** — Laissez la respiration vous guider
3. **Modifiez selon vos besoins** — Utilisez des supports
4. **Pas de précipitation** — Restez le temps que vous voulez
5. **Écoutez votre corps** — Ajustez la position
