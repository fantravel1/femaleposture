# Chien-Oiseau: Exercice de Stabilité du Core

Le chien-oiseau est un exercice fondamental qui renforce le core, les fessiers et le dos tout en défiant l'équilibre et la coordination.

## Données Rapides

| | |
|---|---|
| **Cible** | Core, fessiers, bas du dos, épaules |
| **Idéal Pour** | Stabilité du core, douleur dorsale, équilibre |
| **Difficulté** | Débutant |
| **Équipement** | Tapis (optionnel) |
| **Temps** | 3-5 minutes |

## Pourquoi Ça Fonctionne

- Renforce les muscles stabilisateurs profonds
- Améliore la coordination entre les côtés du corps
- Entraîne le core sans stresser la colonne
- Développe l'équilibre et la proprioception

## Instructions

### Position de Départ

1. Commencez à quatre pattes
2. Poignets sous épaules, genoux sous hanches
3. Colonne neutre, core légèrement actif
4. Regardez vers le sol

### Le Mouvement

1. **Étendez le bras droit** vers l'avant (pouce vers le haut)
2. **Simultanément étendez la jambe gauche** vers l'arrière
3. Gardez le bassin et les épaules de niveau
4. Tenez 3-5 secondes
5. Revenez à la position initiale avec contrôle
6. Répétez de l'autre côté

### Répétitions

- 8-12 répétitions par côté
- 2-3 séries
- Alternez les côtés

## Erreurs Courantes

- **Cambrer le bas du dos** — Gardez le core actif
- **Faire pivoter les hanches** — Le bassin reste stable
- **Retenir la respiration** — Respirez normalement
- **Aller trop vite** — Le contrôle est clé

## Modifications

### Plus Facile

- Étendez seulement le bras (sans la jambe)
- Étendez seulement la jambe (sans le bras)
- Réduisez le temps de maintien

### Plus Difficile

- Augmentez le maintien à 10 secondes
- Ajoutez de petites pulsations en haut
- Utilisez une surface instable

## Points Clés

1. **La stabilité d'abord** — Ne sacrifiez pas la forme
2. **Mouvement lent et contrôlé** — Pas de balancement
3. **Respirez normalement** — Ne retenez pas l'air
4. **Bassin de niveau** — C'est le plus grand défi
