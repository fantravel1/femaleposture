# Appui sur Coudes Ventral: Position d'Extension Douce

S'allonger sur le ventre sur les coudes est une façon douce d'étendre votre colonne. C'est la même position que la posture du sphinx en yoga—accessible et efficace.

## Données Rapides

| | |
|---|---|
| **Cible** | Extension de la colonne, ouverture de l'avant du corps |
| **Idéal Pour** | Contrecarrer la position assise, travail doux du dos |
| **Difficulté** | Débutant |
| **Équipement** | Tapis ou surface douce |
| **Temps** | 1-5 minutes |
| **Fréquence** | Quotidien |

## Pourquoi Cette Position Aide

### Avantages

- Étend doucement la colonne
- Contrecarre la position fléchie assise
- Ouvre la poitrine
- Passive et relaxante
- Peut être maintenue plus longtemps que les positions actives

### Idéal Pour

- Travailleuses de bureau
- Après une position assise prolongée
- Soin doux du dos
- Lire ou se relaxer productivement

## Comment le Faire

### Configuration

1. Allongez-vous sur le ventre sur surface confortable
2. Placez les coudes sous les épaules
3. Avant-bras à plat sur le sol, parallèles
4. Laissez le bas du corps se relaxer

### Position

- Coudes directement sous les épaules
- Avant-bras supportent le haut du corps
- Hanches et jambes restent au sol
- Tête neutre ou regardant vers l'avant

### Maintien

- Respirez normalement
- Laissez le corps se relaxer dans la position
- Maintenez 1-5 minutes
- Descendez quand fatigué

## Points Clés

### Signaux de Forme

- Épaules éloignées des oreilles
- Ne haussez pas
- Respirez dans la poitrine
- Relaxez autant que possible

### Erreurs Courantes

- Coudes trop en avant
- Épaules voûtées
- Retenir sa respiration
- Se forcer pour rester en haut

## Qui Doit Être Prudent

### Peut Ne Pas Convenir Si

- La douleur du bas du dos augmente dans cette position
- Antécédents de problèmes de disque (vérifiez avec votre praticien)
- Incapable de s'allonger sur le ventre
- Grossesse (stades avancés)

### Modifications

- Plus bas: Appui seulement sur les avant-bras, poitrine plus basse
- Avec coussin sous la poitrine
- Maintiens brefs seulement

## Quand Utiliser

### Bons Moments

- Après être restée assise
- Matin au réveil (doux)
- Relaxation du soir
- En lisant ou regardant quelque chose
- Comme position de repos entre les exercices

### Durée

- Commencez avec 30-60 secondes
- Progressez à plusieurs minutes
- Descendez si inconfortable

## Respiration dans Cette Position

### Utilisez l'Opportunité

- La poitrine est ouverte
- Pratiquez la respiration profonde
- Sentez la cage thoracique s'étendre
- Respirations relaxantes

## Progression

### Depuis l'Appui sur Coudes

Vous pouvez progresser vers:
- Press-ups (extension McKenzie)
- Posture du cobra (mains poussant vers le haut)
- Exercices de dos plus actifs
