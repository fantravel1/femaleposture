# Extension Thoracique: Mobilité du Milieu du Dos

L'extension thoracique est essentielle pour contrebalancer la posture affaissée et améliorer la mobilité de la partie médiane de la colonne.

## Données Rapides

| | |
|---|---|
| **Cible** | Colonne thoracique, muscles extenseurs |
| **Idéal Pour** | Cyphose, épaules arrondies, raideur |
| **Difficulté** | Débutant |
| **Équipement** | Rouleau en mousse ou serviette roulée |
| **Temps** | 2-3 minutes |

## Sur Rouleau en Mousse

### Instructions

1. Allongez-vous sur le dos
2. Rouleau perpendiculaire à la colonne, au niveau du milieu du dos
3. Soutenez la tête avec les mains
4. Pieds à plat au sol, genoux pliés
5. Laissez le dos s'étendre sur le rouleau
6. Respirez profondément
7. Déplacez le rouleau vers le haut ou le bas pour différents niveaux
8. Répétez 5 extensions à chaque niveau

## Sur Chaise

### Instructions

1. Asseyez-vous avec le milieu du dos contre le dossier
2. Entrelacez les mains derrière la tête
3. Étendez le dos sur le dossier
4. Regardez vers le plafond
5. Tenez 5 secondes
6. Revenez
7. Répétez 10 fois

## Ce Que Vous Devriez Sentir

- Étirement dans le haut de l'abdomen
- Ouverture de la poitrine
- Mobilité dans le milieu du dos
- Pas de douleur dans le bas du dos

## Erreurs Courantes

- **Étendre depuis le bas du dos** — Concentrez-vous sur la partie médiane
- **Forcer trop** — Graduel et doux
- **Retenir la respiration** — Respirez profondément

## Points Clés

1. **Concentrez-vous sur le milieu du dos** — Pas le bas du dos
2. **Mouvement contrôlé** — Pas de rebonds
3. **Respirez** — Aide à approfondir
4. **Régularité** — Mieux fréquent
