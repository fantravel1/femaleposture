# Coquillage (Clamshell): Renforce le Moyen Fessier

Le coquillage est un exercice fondamental pour renforcer le moyen fessier, crucial pour la stabilité de hanche et la posture.

## Données Rapides

| | |
|---|---|
| **Cible** | Moyen fessier, rotateurs externes de hanche |
| **Idéal Pour** | Stabilité de hanche, douleur au genou, posture |
| **Difficulté** | Débutant |
| **Équipement** | Tapis, bande élastique (optionnel) |
| **Temps** | 3-5 minutes |

## Pourquoi Ça Fonctionne

- Isole le moyen fessier
- Améliore la stabilité du bassin
- Aide à prévenir l'effondrement du genou
- Renforce sans charger les articulations

## Instructions

### Position de Départ

1. Allongez-vous sur le côté
2. Hanches empilées, genoux pliés à 45°
3. Talons ensemble, alignés avec les fessiers
4. Tête appuyée (sur le bras ou un oreiller)
5. Core légèrement actif

### Le Mouvement

1. Gardez les pieds ensemble
2. Ouvrez le genou supérieur vers le plafond
3. Faites pivoter depuis la hanche, pas le dos
4. Gardez le bassin stable (ne pas rouler en arrière)
5. Ouvrez jusqu'où vous pouvez sans bouger le bassin
6. Descendez avec contrôle
7. Répétez

### Répétitions

- 15-20 répétitions par côté
- 2-3 séries
- Focus sur la qualité, pas la vitesse

## Erreurs Courantes

- **Rouler le bassin en arrière** — Le bassin reste stable
- **Bouger depuis le dos** — Le mouvement est dans la hanche
- **Aller trop vite** — Le contrôle est clé
- **Oublier le core** — Gardez l'abdomen légèrement actif

## Variations

### Avec Bande Élastique

- Placez une bande au-dessus des genoux
- Augmente la résistance
- Plus d'activation

### Coquillage Inversé

- Pieds séparés, genoux ensemble
- Ouvrez les pieds vers le haut
- Travaille différentes fibres

### Coquillage Élevé

- Levez les deux genoux légèrement du sol
- Ajoute de l'instabilité
- Plus difficile

## Points Clés

1. **Bassin stable** — L'erreur la plus courante est de pivoter
2. **Sentez le fessier** — Si vous ne le sentez pas, ajustez
3. **Qualité plutôt que quantité** — Mieux moins et bien faits
4. **Les deux côtés** — Travaillez les deux également
5. **Progressez avec bande** — Quand c'est facile
