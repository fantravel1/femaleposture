# Planche Latérale: Renforce le Core Latéral

La planche latérale est excellente pour renforcer les obliques et développer la stabilité latérale.

## Données Rapides

| | |
|---|---|
| **Cible** | Obliques, moyen fessier, épaules |
| **Idéal Pour** | Stabilité latérale, équilibre, core |
| **Difficulté** | Intermédiaire |
| **Équipement** | Tapis |
| **Temps** | 2-3 minutes |

## Instructions

1. Allongez-vous sur le côté
2. Appuyez-vous sur l'avant-bras (coude sous l'épaule)
3. Jambes tendues, pieds empilés
4. Levez les hanches du sol
5. Créez une ligne droite de la tête aux pieds
6. Tenez 15-30 secondes
7. Changez de côté

## Erreurs Courantes

- **Hanches affaissées** — Gardez la ligne droite
- **Pivoter vers l'avant** — Poitrine ouverte
- **Épaule enfoncée** — Poussez le sol

## Modifications

### Plus Facile

- Genoux pliés, appuyée sur les genoux
- Réduisez le temps

### Plus Difficile

- Levez la jambe supérieure
- Ajoutez un mouvement de hanche
- Bras supérieur vers le plafond

## Points Clés

1. **Ligne droite** — De la tête aux pieds
2. **Core actif** — Ne laissez pas tomber les hanches
3. **Épaule stable** — Ne vous enfoncez pas
4. **Les deux côtés** — Travaillez également
