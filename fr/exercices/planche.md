# Planche: Exercice Fondamental du Core

La planche est l'un des exercices les plus efficaces pour renforcer le core. Elle travaille plusieurs muscles simultanément dans une position isométrique sûre.

## Données Rapides

| | |
|---|---|
| **Cible** | Core complet, épaules, dos |
| **Idéal Pour** | Force du core, stabilité, posture |
| **Difficulté** | Débutant à intermédiaire |
| **Équipement** | Aucun |
| **Temps** | 20-60 secondes |

## Pourquoi Ça Fonctionne

- Active tous les muscles du core simultanément
- Position sûre pour la colonne
- Ne nécessite pas de mouvement ni d'équipement
- Facile à progresser

## Instructions

### Position

1. Commencez en position de pompe
2. Avant-bras au sol (ou mains tendues)
3. Coudes directement sous les épaules
4. Corps en ligne droite de la tête aux talons
5. Core actif, fessiers serrés

### Maintenez la Position

- Respirez normalement
- Gardez la ligne droite
- Ne laissez pas tomber les hanches
- Ne montez pas trop les hanches
- Regardez vers le sol

### Temps

- **Débutant:** 10-20 secondes
- **Intermédiaire:** 30-45 secondes
- **Avancé:** 60+ secondes

## Erreurs Courantes

- **Hanches affaissées** — Activez plus le core
- **Hanches trop hautes** — Baissez légèrement
- **Tête tombante** — Regardez le sol, cou aligné
- **Épaules haussées** — Poussez le sol, épaules basses
- **Retenir la respiration** — Respirez continuellement

## Variations

### Planche sur les Genoux

- Genoux au sol
- Gardez la ligne droite de la tête aux genoux
- Idéal pour débutants

### Planche Haute

- Sur les mains tendues
- Bras droits
- Poignets sous les épaules

### Planche Latérale

- Appuyée sur un avant-bras
- Corps sur le côté
- Travaille les obliques

## Progression

1. Commencez par planche sur genoux
2. Progressez vers planche complète 20 secondes
3. Augmentez le temps graduellement
4. Ajoutez des variations

## Points Clés

1. **Qualité plutôt que quantité** — Meilleure forme que plus de temps
2. **Ligne droite** — De la tête aux talons
3. **Core actif** — Nombril vers la colonne
4. **Respirez** — Ne retenez pas l'air
5. **Progressez graduellement** — Ne forcez pas le temps
