# Squats: Renforce Jambes et Fessiers

Le squat est l'un des exercices les plus fondamentaux et efficaces pour renforcer le bas du corps.

## Données Rapides

| | |
|---|---|
| **Cible** | Quadriceps, fessiers, ischio-jambiers, core |
| **Idéal Pour** | Force générale, posture, fonction quotidienne |
| **Difficulté** | Débutant à avancé |
| **Équipement** | Aucun (poids optionnel) |
| **Temps** | 5 minutes |

## Instructions

### Position de Départ

1. Pieds largeur d'épaules (ou légèrement plus)
2. Orteils légèrement vers l'extérieur
3. Poids sur tout le pied
4. Core actif, poitrine relevée

### Le Mouvement

1. Poussez les hanches vers l'arrière
2. Pliez les genoux
3. Descendez comme si vous vous asseyiez sur une chaise
4. Gardez les genoux au-dessus des orteils
5. Descendez aussi loin que possible avec bonne forme
6. Poussez à travers les talons pour monter
7. Serrez les fessiers en haut

### Répétitions

- 12-15 répétitions
- 2-3 séries

## Erreurs Courantes

- **Genoux vers l'intérieur** — Poussez vers l'extérieur
- **Talons qui se lèvent** — Gardez les pieds plantés
- **Dos arrondi** — Gardez la poitrine haute
- **Ne pas descendre assez** — Visez parallèle ou plus

## Variations

### Squat avec Chaise
- Descendez jusqu'à toucher une chaise
- Bon pour apprendre

### Squat Sumo
- Pieds plus larges, orteils plus vers l'extérieur
- Plus d'emphase sur les adducteurs

### Squat Goblet
- Tenez un poids à la poitrine
- Aide à la forme

## Points Clés

1. **Poids dans les talons** — Pas sur les orteils
2. **Genoux vers l'extérieur** — Qu'ils ne s'effondrent pas
3. **Poitrine haute** — Dos neutre
4. **Profondeur sûre** — Où vous maintenez bonne forme
