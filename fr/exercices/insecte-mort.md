# Insecte Mort (Dead Bug): Stabilité du Core

L'insecte mort apprend à votre core à maintenir la stabilité pendant que les membres bougent. Il est sûr pour le bas du dos et excellent pour la posture.

## Données Rapides

| | |
|---|---|
| **Cible** | Core profond, coordination |
| **Idéal Pour** | Stabilité, douleur dorsale, posture |
| **Difficulté** | Débutant |
| **Équipement** | Tapis |
| **Temps** | 3-5 minutes |

## Pourquoi Ça Fonctionne

- Entraîne le core dans sa vraie fonction (stabiliser)
- Sûr pour la colonne
- Améliore la coordination
- Transfère aux mouvements de la vie quotidienne

## Instructions

### Position de Départ

1. Allongez-vous sur le dos
2. Bras tendus vers le plafond
3. Hanches et genoux à 90°
4. Bas du dos légèrement pressé contre le sol
5. Core actif

### Le Mouvement

1. Expirez et activez le core
2. Étendez le bras droit vers l'arrière
3. Simultanément étendez la jambe gauche
4. Gardez le bas du dos en contact avec le sol
5. Revenez avec contrôle
6. Répétez de l'autre côté
7. Alternez

### Répétitions

- 8-12 répétitions par côté
- 2-3 séries
- Concentrez-vous sur ne pas cambrer le dos

## Erreurs Courantes

- **Cambrer le bas du dos** — L'erreur la plus courante; réduisez l'amplitude
- **Retenir la respiration** — Expirez en étendant
- **Bouger trop vite** — Le contrôle est essentiel
- **Perdre la position neutre** — Gardez le dos stable

## Variations

### Plus Facile

- Étendez seulement les jambes (bras immobiles)
- Étendez seulement les bras (jambes immobiles)
- Réduisez l'amplitude d'extension

### Plus Difficile

- Ajoutez une pause en extension
- Ralentissez le mouvement
- Étendez complètement la jambe jusqu'au sol

## Points Clés

1. **Le bas du dos est la priorité** — Ne doit pas se cambrer
2. **Expirez en étendant** — Aide à maintenir le core
3. **Lent et contrôlé** — Ce n'est pas une course
4. **Opposés ensemble** — Bras droit avec jambe gauche
5. **Progressez graduellement** — Augmentez l'amplitude avec le temps
