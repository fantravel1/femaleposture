# Pont Fessier: Renforce Fessiers et Core

Le pont fessier est un exercice fondamental qui active et renforce les fessiers, essentiels pour la posture et la stabilité pelvienne.

## Données Rapides

| | |
|---|---|
| **Cible** | Fessiers, ischio-jambiers, core, bas du dos |
| **Idéal Pour** | Activation des fessiers, douleur dorsale, posture |
| **Difficulté** | Débutant |
| **Équipement** | Tapis (optionnel) |
| **Temps** | 3-5 minutes |

## Pourquoi Ça Fonctionne

- Active les fessiers qui peuvent être « endormis » à force d'être assis
- Renforce sans charger la colonne
- Contrebalance les effets de la position assise
- Améliore la stabilité pelvienne

## Instructions

### Position de Départ

1. Allongez-vous sur le dos
2. Genoux pliés, pieds à plat sur le sol
3. Pieds écartés largeur de hanches
4. Bras sur les côtés, paumes vers le bas
5. Colonne neutre

### Le Mouvement

1. Poussez à travers les talons
2. Levez les hanches du sol
3. Serrez les fessiers en haut
4. Créez une ligne droite des épaules aux genoux
5. Tenez 2-3 secondes
6. Descendez avec contrôle
7. Répétez

### Répétitions

- 12-15 répétitions
- 2-3 séries
- Repos bref entre les séries

## Erreurs Courantes

- **Cambrer le dos** — Le mouvement vient des hanches
- **Pousser avec les orteils** — Poussez avec les talons
- **Ne pas serrer les fessiers** — Contractez en haut
- **Monter trop haut** — N'hyperétendez pas

## Variations

### Pont avec Pause

- Tenez en haut 5-10 secondes
- Plus d'activation

### Pont Unipodal

- Une jambe tendue
- Beaucoup plus difficile
- Travaille l'équilibre

### Pont avec Bande

- Bande élastique sur les genoux
- Poussez vers l'extérieur contre la bande
- Plus d'activation du moyen fessier

## Points Clés

1. **Sentez les fessiers** — Si vous ne les sentez pas, ajustez
2. **Poussez par les talons** — Active plus les fessiers
3. **Contrôle en descendant** — Ne vous laissez pas tomber
4. **Respirez** — Expirez en montant
5. **Progressez graduellement** — Ajoutez des variations
