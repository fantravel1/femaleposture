# Trampoline Posture

Trampoline exercise (rebounding) offers low-impact cardio. Proper posture protects your joints and maximizes benefits.

## Basic Position

### Standing on Trampoline

**Starting stance:**
- Feet hip-width apart
- Knees soft
- Core engaged
- Shoulders relaxed
- Eyes forward

### Bounce Position

**While rebounding:**
- Upright torso
- Controlled bounce
- Arms for balance
- Soft landings
- Centered position

## Bounce Technique

### Basic Bounce

**Health bounce:**
- Feet stay on surface
- Gentle up and down
- Core engagement
- Minimal height
- Low impact

### Full Bounce

**Leaving surface:**
- Controlled height
- Soft landings
- Arms for balance
- Centered position
- Core stability

## Safety Considerations

### Balance

**Staying safe:**
- Start slowly
- Hold bar if available
- Build balance gradually
- Know limits
- Stop if dizzy

### Landing

**Protecting joints:**
- Land softly
- Knees absorb
- Centered on mat
- Controlled descent
- Avoid edge

## Women-Specific Considerations

### Pelvic Floor

**Special attention:**
- Awareness needed
- Modifications available
- Build gradually
- Address issues
- Professional guidance if needed

Good trampoline posture creates safe, effective rebounding exercise with proper joint protection.
