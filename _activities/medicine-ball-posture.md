# Medicine Ball Posture

Medicine balls add dynamic challenge to workouts. Proper posture protects your back during throwing and rotational movements.

## Basic Handling

### Holding Position

**Ball grip:**
- Hands on sides
- Arms comfortable
- Ball close to body
- Core engaged
- Balanced stance

### Weight Selection

**Appropriate ball:**
- Controllable weight
- Proper form possible
- Challenge without compensation
- Progressive increase
- Exercise-specific

## Throwing Exercises

### Overhead Throw

**Slam technique:**
- Reach overhead
- Core tight
- Forceful throw down
- Controlled movement
- Hip hinge to catch

### Chest Pass

**Push technique:**
- Ball at chest
- Elbows out
- Extend forcefully
- Follow through
- Ready for return

## Rotational Work

### Twists

**Core rotation:**
- Feet grounded
- Core drives rotation
- Arms follow torso
- Controlled speed
- Equal both sides

### Side Throws

**Lateral power:**
- Athletic stance
- Rotate from core
- Release at side
- Control return
- Balanced effort

## Floor Exercises

### Core Work

**Abdominal exercises:**
- Ball adds challenge
- Proper form priority
- Core engagement
- Controlled movement
- Breathing pattern

Good medicine ball posture uses the core effectively while protecting the spine during dynamic movements.
