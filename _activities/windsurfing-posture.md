# Windsurfing Posture

Windsurfing requires balance and sail control. Proper posture enables efficient sailing.

## Basic Stance

### On Board

**Standing position:**
- Feet shoulder-width
- Knees bent
- Hips forward
- Core engaged
- Weight centered

### Sailing Position

**Moving forward:**
- Back foot in strap
- Mast foot position key
- Arms extended
- Shoulders back
- Eyes forward

## Sail Handling

### Uphaul

**Getting started:**
- Knees bent
- Back straight
- Pull with legs
- Gradual lift
- Balance maintained

### Sheeting In

**Catching wind:**
- Back hand pulls
- Front hand guides
- Progressive power
- Body adjusts
- Speed builds

## Power Control

### Harnessing Wind

**Efficient sailing:**
- Harness hook engaged
- Arms relaxed
- Hips absorb gusts
- Core stable
- Sustainable position

### Gusts

**Managing power:**
- Sheet out quickly
- Knees absorb
- Core engaged
- Recover position
- Resume sailing

## Turning

### Basic Tack

**Turning through wind:**
- Step around mast
- Flip sail
- Quick transition
- New stance
- Resume sailing

### Jibe

**Downwind turn:**
- Weight shift
- Sail flip
- Board continues
- Smooth transition
- Controlled motion

## Harness Technique

### Using Harness

**Reducing arm fatigue:**
- Hook in smoothly
- Hips take load
- Arms relaxed
- Core engaged
- Sustainable effort

Good windsurfing posture creates efficient sail handling and comfortable riding.
