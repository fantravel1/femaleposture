# Hotel Sleep Posture

Hotel beds differ from home. Proper positioning enables good sleep away from home.

## Bed Assessment

### Mattress Firmness

**Evaluating:**
- Softer or firmer than home
- Adjust as possible
- Extra pillows help
- Topper request option
- Adaptation needed

### Pillow Options

**Available support:**
- Multiple pillows usually
- Request extras
- Build support
- Various heights
- Personal preference

## Sleep Position

### Side Sleeping

**In hotel:**
- Pillow between knees
- Proper head pillow
- Neutral spine
- Comfortable arms
- Consistent position

### Back Sleeping

**On hotel bed:**
- Pillow under knees
- Neck supported
- Arms comfortable
- Spine neutral
- Relaxed position

## Adaptation Strategies

### First Night

**Adjusting:**
- Give yourself time
- Arrange pillows
- Find comfort
- Travel fatigue helps
- Accept differences

### Multiple Nights

**Longer stays:**
- Consistency helps
- Request room adjustments
- Create routine
- Improve each night
- Maintain habits

## Common Issues

### Too Soft

**Sinking mattress:**
- Pillows for support
- Sleep on edge
- Request firmer if possible
- Extra bedding under sheet
- Manage as able

### Too Firm

**Hard surface:**
- Extra bedding
- Pillows strategically
- May prefer
- Soften with topper
- Adapt position

## Travel Fatigue

### Jet Lag Impact

**Sleep disruption:**
- Adjust schedule
- Light exposure
- Physical activity
- Give time
- Healthy sleep habits

Good hotel sleep posture enables rest during travel.
