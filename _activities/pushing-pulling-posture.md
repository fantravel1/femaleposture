# Pushing and Pulling Posture

Pushing and pulling moves objects. Proper technique protects your body during force application.

## Pushing Technique

### Basic Push

**Forward force:**
- Face object
- Hands at comfortable height
- Elbows slightly bent
- Push from legs
- Core engaged

### Heavy Objects

**More force needed:**
- Low hand position
- Lean into push
- Drive with legs
- Short bursts
- Check path first

## Pulling Technique

### Basic Pull

**Backward force:**
- Face away from movement
- Grip securely
- Use legs
- Walk backward
- Watch where going

### When to Pull

**Pull considerations:**
- Generally pushing preferred
- Pull for specific situations
- Careful of tripping
- Clear path
- Controlled motion

## Doors

### Push Doors

**Opening away:**
- Hand on door
- Push with arm
- Walk through
- Controlled motion
- Gentle force

### Pull Doors

**Opening toward:**
- Firm grip
- Step back
- Pull with arm
- Create space
- Walk through

## Wheeled Objects

### Carts and Strollers

**Rolling items:**
- Push usually better
- Upright posture
- Steady pace
- Watch path
- Controlled motion

### Luggage

**Wheeled bags:**
- Pull behind
- Don't twist
- Alternate hands
- Manage weight
- Smooth surfaces easier

Good pushing and pulling posture applies force safely and effectively.
