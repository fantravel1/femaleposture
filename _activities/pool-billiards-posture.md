# Pool Billiards Posture

Pool requires precise positioning. Proper posture enables accurate, consistent shots.

## Stance

### Foot Position

**At the table:**
- Feet shoulder-width
- Front foot toward shot
- Weight balanced
- Stable base
- Comfortable stance

### Body Angle

**Bending to table:**
- Bend at hips
- Back relatively straight
- Eyes over cue
- Comfortable position
- Consistent approach

## Bridge Hand

### Open Bridge

**Basic bridge:**
- Hand flat on table
- Thumb and finger guide
- Stable support
- Common bridge
- Good for beginners

### Closed Bridge

**Secure loop:**
- Finger loop around cue
- More control
- Advanced technique
- Stable stroke
- Personal preference

## Grip Hand

### Holding Cue

**Rear hand:**
- Relaxed grip
- Elbow at 90 degrees
- Pendulum motion
- Consistent position
- Smooth stroke

### Stroke

**The motion:**
- Pendulum from elbow
- Smooth acceleration
- Follow through
- Consistent stroke
- Practice refines

## Head Position

### Eye Alignment

**Sighting:**
- Chin over cue
- Eyes on target
- Consistent position
- Focus on aim
- Developed feel

## Shot Execution

### Pre-Shot Routine

**Consistency:**
- Same approach
- Practice strokes
- Visualization
- Committed execution
- Repeat process

Good pool posture enables accurate, consistent cue sport play.
