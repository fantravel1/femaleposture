# Rowing Machine Posture

The rowing machine provides full-body exercise. Proper posture maximizes benefits and prevents injury.

## The Catch

### Starting Position

**Ready to drive:**
- Shins vertical
- Arms straight
- Core engaged
- Shoulders over hips
- Prepared for push

### Grip

**Holding handle:**
- Relaxed hands
- Over-grip
- Not too tight
- Wrists flat
- Connected to body

## The Drive

### Leg Push

**Power phase:**
- Drive through heels
- Legs extend
- Back stays stable
- Arms straight initially
- Sequential power

### Sequence

**Proper order:**
- Legs first
- Back opens
- Arms finish
- Smooth motion
- Connected movement

## The Finish

### End Position

**Completed stroke:**
- Legs straight
- Slight back lean
- Handle to lower ribs
- Elbows back
- Momentary pause

## The Recovery

### Return

**Going back:**
- Arms extend first
- Body hinges forward
- Knees bend
- Controlled motion
- Prepare for catch

### Speed

**Recovery pace:**
- Slower than drive
- Controlled return
- No rushing
- Rest phase
- 1:2 ratio roughly

## Common Errors

### To Avoid

**Technique flaws:**
- Rushing recovery
- Chicken wings
- Back too early
- Grip too tight
- Incomplete leg drive

Good rowing machine posture creates efficient, powerful strokes for maximum benefit.
