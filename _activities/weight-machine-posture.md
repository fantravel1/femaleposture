# Weight Machine Posture

Weight machines guide movement but still require proper positioning. Correct setup protects joints and targets intended muscles.

## Machine Setup

### Seat Adjustment

**Proper positioning:**
- Adjust for your height
- Pivot points align with joints
- Comfortable range of motion
- Pad positioning
- Read machine instructions

### Weight Selection

**Starting point:**
- Conservative start
- Proper form possible
- Controlled movement
- Progressive increase
- No ego lifting

## General Principles

### Body Alignment

**During exercise:**
- Back supported when appropriate
- Core engaged
- Neutral spine
- Shoulders set
- Controlled breathing

### Movement Quality

**Proper execution:**
- Full range of motion
- Controlled speed
- No momentum
- Smooth motion
- Both phases count

## Common Machines

### Chest Press

**Proper form:**
- Back flat against pad
- Feet grounded
- Handles at chest level
- Full extension
- Controlled return

### Lat Pulldown

**Proper form:**
- Sit tall
- Thighs secured
- Pull to chest level
- Shoulder blades engage
- Controlled return

### Leg Press

**Proper form:**
- Full back contact
- Feet hip-width
- Knees track over toes
- Don't lock knees
- Controlled movement

## Safety

### Awareness

**Safe machine use:**
- Know the machine
- Adjust first
- Test with light weight
- Watch for pinch points
- Full control always

Good weight machine posture ensures proper muscle targeting and joint protection during resistance training.
