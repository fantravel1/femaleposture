# Package Receiving Posture

Package receiving involves unexpected lifting. Proper posture prevents strain when handling deliveries.

## Picking Up Packages

### Safe Lifting

**From ground:**
- Squat down
- Assess weight first
- Grip securely
- Back straight
- Lift with legs

### Heavy Packages

**Heavier items:**
- Get help
- Use dolly
- Drag if needed
- Request placement
- Safe handling

## Carrying Inside

### Box Transport

**Moving package:**
- Close to body
- See over package
- Clear path
- Set down safely
- Controlled carrying

### Multiple Packages

**Several deliveries:**
- Multiple trips
- Manageable loads
- Don't stack too high
- Safe transport
- Efficient moving

## Opening Packages

### Unpacking

**Box contents:**
- Set on surface
- Open comfortably
- Don't hunch over
- Easy access
- Comfortable position

## Box Disposal

### Breakdown

**Recycling:**
- Break down boxes
- Comfortable position
- Stomp flat
- Bundle for recycling
- Complete disposal

## Delivery Instructions

### Prevent Lifting

**Future deliveries:**
- Request placement
- Gate codes
- Specific instructions
- Reduce lifting
- Easier receiving

Good package receiving posture prevents strain during delivery handling.
