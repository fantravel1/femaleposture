# Getting Up Posture

Getting up from seated happens often. Proper posture supports safe, efficient rising.

## From Chair

### Standard Rising

**Chair to standing:**
- Feet under body
- Lean forward slightly
- Push through legs
- Rise controlled
- Stand tall

### Using Arms

**Arm assistance:**
- Hands on armrests
- Or hands on thighs
- Push to assist
- Controlled rise
- Safe transition

## From Low Surfaces

### Couch Rising

**Lower seats:**
- Scoot forward
- Feet firmly planted
- Core engaged
- Push to stand
- Controlled motion

### Floor Rising

**From ground:**
- Roll to side
- Push to kneeling
- One foot up
- Rise from there
- Safe progression

## Leg Strength

### Muscle Use

**Rising power:**
- Legs do work
- Core engaged
- Arms assist if needed
- Controlled motion
- Strength benefit

## Multiple Times

### Frequent Rising

**Throughout day:**
- Consistent technique
- Good exercise
- Functional movement
- Safe practice
- Daily occurrence

## Balance Consideration

### Safe Rising

**Careful transition:**
- Pause if dizzy
- Secure footing
- Use support if needed
- Careful standing
- Safe movement

Good getting up posture supports safe, frequent sit-to-stand transitions.
