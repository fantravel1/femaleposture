# Woodworking Posture

Woodworking combines standing, bending, and tool use that can strain the body. Proper posture protects you while enabling quality craftsmanship.

## Bench Work

### Standing Position

**At the bench:**
- Bench at proper height
- Weight balanced
- Core engaged
- Arms at comfortable level
- Anti-fatigue mat

### Work Height

**Task matching:**
- Different heights for different tasks
- Risers when needed
- Bending minimized
- Comfortable reach
- Adjustable options

## Tool Use

### Hand Tools

**Safe technique:**
- Proper grip
- Body position
- Let tool do work
- Core stability
- Safe stance

### Power Tools

**Machine posture:**
- Stable stance
- Safe distance
- Proper height
- Body aligned
- Safety awareness

## Lifting and Carrying

### Material Handling

**Safe practices:**
- Proper lifting technique
- Get help when needed
- Use carts and dollies
- Know limits
- Safe paths

## Common Issues

### Back Strain

**From bending:**
- Proper bench height
- Core engagement
- Position variety
- Regular breaks
- Stretching routine

### Shoulder Problems

**From tool use:**
- Proper technique
- Tool sharpness
- Rest periods
- Strength building
- Warm-up routine

Good woodworking posture combines safety and efficiency, protecting your body through projects large and small.
