# Arm Exercises Posture

Arm exercises require stable positioning. Proper posture ensures effective, safe upper body training.

## Bicep Exercises

### Bicep Curls

**Curl technique:**
- Elbows at sides
- Core engaged
- No swinging
- Controlled movement
- Full range

### Form Focus

**Avoiding mistakes:**
- Stable torso
- No momentum
- Controlled speed
- Proper weight
- Quality reps

## Tricep Exercises

### Tricep Extensions

**Overhead or kickback:**
- Stable core
- Elbow position
- Controlled motion
- Full extension
- Safe movement

### Dips

**Body weight dips:**
- Shoulders safe position
- Controlled depth
- Proper support
- Core engaged
- Safe practice

## Push Exercises

### Push-Ups

**Push-up form:**
- Plank position
- Core engaged
- Elbows track properly
- Controlled movement
- Full range

### Overhead Press

**Pressing up:**
- Core braced
- Proper path
- Full extension
- Controlled return
- Safe technique

## Standing vs Seated

### Position Options

**Training position:**
- Standing requires core
- Seated more isolating
- Both valuable
- Personal preference
- Varied training

## Shoulder Safety

### Joint Protection

**Safe training:**
- Warm up properly
- Appropriate range
- Listen to signals
- Proper weight
- Injury prevention

Good arm exercise posture ensures effective, safe upper body training.
