# Climbing Stairs Posture

Stair climbing is daily exercise. Proper posture supports safe, efficient stair navigation.

## Ascending Stairs

### Climbing Up

**Going upstairs:**
- Upright posture
- Use whole foot
- Push through heel
- Core engaged
- Steady pace

### Using Rail

**Handrail use:**
- Light support
- Not pulling up
- Balance assistance
- Safety option
- Available resource

## Descending Stairs

### Coming Down

**Going downstairs:**
- Controlled descent
- See steps
- Use handrail
- Core engaged
- Safe footing

### Taking Care

**Descent safety:**
- No rushing
- Full attention
- Each step secure
- Watch footing
- Careful movement

## Carrying Items

### With Loads

**Carrying up/down:**
- One hand free for rail
- Manageable weight
- Multiple trips
- Safe practice
- Careful navigation

## Exercise Climbing

### Stair Workout

**Fitness stairs:**
- Warm up first
- Proper footwear
- Controlled pace
- Cardiovascular benefit
- Progressive training

## Common Mistakes

### Posture Issues

**Corrections:**
- Don't lean forward
- Eyes ahead not down
- Core engaged
- Controlled pace
- Good form

Good stair climbing posture supports safe, effective daily navigation.
