# Archery Posture

Archery demands precise, consistent body positioning for accuracy. Understanding proper archery posture improves your shooting while preventing strain.

## Basic Stance

### Foot Position

**Foundation:**
- Feet shoulder-width
- Perpendicular to target
- Weight even
- Stable base
- Consistent placement

### Body Alignment

**Posture setup:**
- Upright stance
- Core engaged
- Shoulders level
- Head turned to target
- Relaxed but aligned

## Draw Sequence

### Bow Raise

**Preparation:**
- Grip relaxed
- Arm raises bow
- Shoulder aligned
- Core stable
- Eyes on target

### Draw

**Pulling back:**
- Use back muscles
- Draw to anchor point
- Consistent anchor
- Relaxed bow hand
- Proper alignment

### Anchor Point

**Reference position:**
- String to corner of mouth
- Finger position consistent
- Head position stable
- Full draw achieved
- Ready for release

## Release and Follow Through

### Release

**Letting go:**
- Relaxed fingers
- Back tension continues
- Smooth release
- No grabbing
- Consistent action

### Follow Through

**After shot:**
- Bow arm stays up
- Draw hand releases back
- Maintain position
- Watch arrow
- Analyze shot

## Women-Specific Considerations

### Physical Factors

**Female archers:**
- Draw weight appropriate
- Bow fit
- Chest clearance
- Arm length considerations
- Strength building

### Equipment

**Proper setup:**
- Bow sizing
- Arrow length
- Draw weight progression
- Protective gear
- Comfortable practice

Good archery posture creates the consistency necessary for accurate shooting every time.
