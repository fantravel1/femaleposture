# Core Exercises Posture

Core exercises require precise form. Proper posture ensures effective, safe abdominal training.

## Lying Core Work

### Supine Position

**On back:**
- Lower back position
- Neutral or imprinted
- Head position
- Breathing pattern
- Controlled movement

### Crunches

**Basic crunch:**
- Hands support head
- Lift shoulder blades
- Core contracts
- No neck strain
- Controlled motion

## Plank Positions

### Standard Plank

**Prone hold:**
- Straight line body
- Core engaged
- No sagging
- No piking
- Proper duration

### Side Plank

**Lateral hold:**
- Stacked alignment
- Core engaged
- Hip lifted
- Stable position
- Both sides equal

## Bird Dog

### Opposite Limb

**Quadruped position:**
- Neutral spine
- Core engaged
- Controlled extension
- Balanced movement
- Alternating sides

## Dead Bug

### Controlled Movement

**Supine work:**
- Lower back pressed
- Opposite arm and leg
- Controlled motion
- Core engagement
- Safe technique

## Rotation Exercises

### Trunk Rotation

**Rotational core:**
- Controlled movement
- Core engaged
- Spine protected
- Balanced work
- Functional strength

## Breathing

### During Core Work

**Breath pattern:**
- Exhale on effort
- Don't hold breath
- Controlled breathing
- Consistent pattern
- Effective training

Good core exercise posture ensures safe, effective trunk strengthening.
