# Gymnastics Posture

Gymnastics demands precise body control and alignment. Proper posture is fundamental to every skill and apparatus.

## Basic Positions

### Straight Body

**Hollow position:**
- Core engaged
- Lower back pressed flat
- Arms by ears
- Pointed toes
- Strong line

### Arch Position

**Extension:**
- Controlled arch
- Core still engaged
- Shoulders back
- Full body tension
- Safe range

## Floor Exercise

### Tumbling

**Flipping and twisting:**
- Set position
- Core control
- Spatial awareness
- Landing posture
- Safe technique

### Dance Elements

**Artistic movements:**
- Elegant carriage
- Pointed extremities
- Core strength
- Balance and grace
- Expression

## Balance Beam

### Walking

**Basic skills:**
- Core engagement
- Focused gaze
- Precise foot placement
- Arms for balance
- Controlled movement

### Skills

**Beam elements:**
- Same technique as floor
- Heightened precision
- Balance priority
- Safe dismount
- Core stability

## Uneven Bars

### Swinging

**Bar skills:**
- Hollow body
- Tight form
- Core engagement
- Shoulder stability
- Hand grip

## Vault

### Approach

**Run and hurdle:**
- Speed building
- Proper hurdle
- Board contact
- Powerful takeoff
- Flight position

Good gymnastics posture is the foundation of every skill, creating the body control necessary for safe, beautiful movement.
