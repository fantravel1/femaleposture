# Calisthenics Posture

Calisthenics uses bodyweight for strength training. Proper posture ensures effective, safe movement.

## Basic Positions

### Push Position

**Pushing exercises:**
- Straight body line
- Core engaged
- Shoulder stability
- Wrist alignment
- Controlled movement

### Pull Position

**Pulling exercises:**
- Active shoulders
- Core tight
- Controlled swing
- Full range
- Grip strength

## Foundational Exercises

### Push-Ups

**Proper form:**
- Straight line
- Elbows tracking
- Core tight
- Full range
- Controlled tempo

### Pull-Ups

**Bar work:**
- Dead hang start
- Full pull
- Core engaged
- Controlled descent
- Multiple grip options

### Dips

**Parallel bars:**
- Shoulder position
- Core tight
- Controlled descent
- Full extension
- Progressive depth

### Squats

**Leg work:**
- Feet position
- Depth appropriate
- Knees tracking
- Core engaged
- Upright torso

## Skill Work

### Handstands

**Inverted balance:**
- Straight body
- Active shoulders
- Core tight
- Wall progression
- Freestanding goal

### Muscle-Ups

**Advanced pull:**
- Explosive pull
- Transition technique
- Core involvement
- Full extension
- Progressive training

## Training Approach

### Progressive Overload

**Building strength:**
- Proper form always
- Progressive difficulty
- Adequate rest
- Consistent training
- Patience with skills

Good calisthenics posture creates the body control necessary for increasingly impressive bodyweight skills.
