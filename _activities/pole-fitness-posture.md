# Pole Fitness Posture

Pole fitness demands strength, flexibility, and body control. Proper posture ensures safe, graceful movement.

## Basic Grip

### Hand Position

**Grip technique:**
- Full grip on pole
- Wrist neutral
- Arm engaged
- Shoulder stable
- Core connected

### Body Position

**On the pole:**
- Core engaged
- Shoulders stable
- Controlled movement
- Balanced position
- Safe grip

## Spins

### Entry

**Starting spins:**
- Controlled approach
- Grip established
- Core engaged
- Leg positioning
- Momentum control

### During Spin

**Rotation:**
- Core tight
- Legs positioned
- Graceful movement
- Speed control
- Safe exit

## Climbs

### Climbing Technique

**Going up:**
- Core engagement
- Leg grip
- Arm pulls
- Efficient movement
- Safe descent

## Inverts

### Going Upside Down

**Inversion safety:**
- Strength required first
- Core control crucial
- Proper progression
- Spotter when learning
- Safe return

### Inverted Positions

**Upside down holds:**
- Core extremely engaged
- Multiple contact points
- Controlled breathing
- Safe transitions
- Strength basis

## Progression

### Building Safely

**Training approach:**
- Strength before tricks
- Proper progression
- Core development
- Flexibility work
- Professional instruction

Good pole fitness posture combines strength and grace while prioritizing safety at every skill level.
