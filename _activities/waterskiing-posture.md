# Waterskiing Posture

Waterskiing demands core strength and balance on water. Proper posture enables smooth, controlled skiing.

## Deep Water Start

### Starting Position

**In water:**
- Skis ahead
- Knees bent
- Arms straight
- Core engaged
- Patient start

### Rising

**Getting up:**
- Let boat pull
- Stay compact
- Core tight
- Rise gradually
- Weight back initially

## Riding Position

### Two-Ski Stance

**Basic skiing:**
- Knees bent
- Weight balanced
- Arms relaxed
- Core engaged
- Shoulders back

### Slalom Stance

**Single ski:**
- Back foot in binding
- Knees bent
- Weight centered
- Core very engaged
- Balance demanding

## Cutting

### Cross-Wake

**Edge technique:**
- Edge pressure
- Core stability
- Weight distribution
- Speed control
- Smooth transitions

## Safety

### Falling

**Safe release:**
- Let go of rope
- Cover head
- Relax body
- Signal OK
- Wait for boat

## Women-Specific

### Physical Factors

**Female skiing:**
- Core strength crucial
- Leg strength
- Equipment fit
- Gradual progression
- Fun first

Good waterskiing posture creates the stability for controlled skiing and skill progression.
