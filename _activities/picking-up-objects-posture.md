# Picking Up Objects Posture

Picking up dropped objects is daily occurrence. Proper posture prevents strain during retrieval.

## Light Objects

### Quick Pickup

**Light items:**
- Squat or lunge
- Back straight
- Quick retrieval
- Return to standing
- Efficient motion

### Golfer's Lift

**Single item:**
- One leg back
- Balance on other
- Reach down
- Quick grab
- Return upright

## Heavier Objects

### Squat Lift

**Weightier items:**
- Full squat
- Both hands
- Close to body
- Lift with legs
- Controlled rise

### Assess First

**Before lifting:**
- Check weight
- Plan lift
- Proper technique
- Safe lifting
- Protect back

## Multiple Objects

### Several Items

**Picking up many:**
- Multiple trips
- Gathered collection
- Manageable loads
- Organized approach
- Efficient retrieval

## Frequent Bending

### Throughout Day

**Many pickups:**
- Consistent technique
- Protect back
- Good exercise
- Functional movement
- Daily occurrence

## Small Objects

### Tiny Items

**Small retrieval:**
- Precision needed
- Comfortable position
- Good visibility
- Patient pickup
- Complete task

Good picking up posture protects back during frequent object retrieval.
