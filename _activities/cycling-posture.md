# Cycling Posture

Cycling posture depends heavily on bike fit and riding style. Understanding proper positioning, especially for the female body, prevents discomfort and injury while improving performance.

## Bike Fit Basics

### Key Measurements

**Proper setup:**
- Saddle height
- Saddle fore/aft
- Handlebar height
- Reach distance
- Cleat position

### Women-Specific Fit

**Female considerations:**
- Narrower shoulders
- Wider hips
- Different torso proportion
- Saddle width needs
- Handlebar width

## Riding Position

### Road Cycling

**Proper form:**
- Slight forward lean
- Neutral spine
- Relaxed shoulders
- Bent elbows
- Engaged core

### Mountain Biking

**Trail positioning:**
- More upright
- Ready stance
- Flexible joints
- Weight distribution
- Dynamic balance

### Stationary/Indoor

**Spin class form:**
- Proper bike setup
- Core engagement
- Shoulder relaxation
- Hand position variety
- Breathing focus

## Common Problems

### Discomfort Areas

**Typical issues:**
- Saddle pain
- Neck strain
- Low back ache
- Numb hands
- Knee pain

### Solutions

**Addressing problems:**
- Saddle adjustment
- Position changes
- Strengthening exercises
- Flexibility work
- Professional fit

## Off-Bike Support

### Strength Training

**Supporting cycling:**
- Core stability
- Hip strength
- Upper body balance
- Flexibility work
- Cross-training

Proper cycling posture reduces discomfort, prevents injury, and allows you to ride farther and more enjoyably.
