# Stability Ball Posture

Stability balls challenge balance and core engagement. Proper posture maximizes benefits while preventing falls.

## Ball Selection

### Proper Size

**Right fit:**
- Thighs parallel when seated
- 90-degree knee angle
- Feet flat on floor
- Appropriate for height
- Proper inflation

## Sitting on Ball

### Basic Position

**Ball sitting:**
- Center of ball
- Feet hip-width
- Core engaged
- Spine tall
- Balanced position

### As Chair Alternative

**Ball as seat:**
- Time-limited
- Core engagement
- Feet grounded
- Regular breaks
- Not all-day replacement

## Exercise Positions

### Lying on Ball

**Back on ball:**
- Controlled mounting
- Core engaged
- Feet grounded
- Stable position
- Safe exercises

### Kneeling on Ball

**Advanced position:**
- Build to this
- Core strength required
- Wall nearby
- Controlled movement
- Safety priority

## Common Exercises

### Core Work

**Abdominal exercises:**
- Crunches
- Back extensions
- Planks
- Rollouts
- Pelvic tilts

### Upper Body

**Pressing movements:**
- Chest press on ball
- Rows
- Shoulder work
- Core challenged
- Balanced position

## Safety

### Fall Prevention

**Safe use:**
- Clear space around
- Start basic
- Build progressively
- Know limits
- Spotter for advanced

Good stability ball posture develops core strength and balance while maintaining safety during exercises.
