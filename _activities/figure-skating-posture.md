# Figure Skating Posture

Figure skating combines athletics with artistry. Proper posture creates both beautiful lines and safe technique.

## Basic Skating Posture

### Gliding Position

**On ice:**
- Upright carriage
- Core engaged
- Shoulders level
- Head balanced
- Elegant arms

### Edge Work

**Blade control:**
- Knee bend
- Core stability
- Weight over edge
- Balance maintained
- Smooth transitions

## Jumps

### Takeoff

**Launch position:**
- Edge loading
- Core engaged
- Arms set
- Rotation prep
- Powerful spring

### Air Position

**During rotation:**
- Tight position
- Arms pulled in
- Core tight
- Head aligned
- Controlled rotation

### Landing

**Safe landing:**
- Knee absorbs
- Arms open
- Core engaged
- Balance recovery
- Continued flow

## Spins

### Entry

**Starting position:**
- Edge approach
- Core centered
- Arms positioning
- Balance centered
- Rotation initiation

### Spinning

**Rotation position:**
- Centered axis
- Position variations
- Core stability
- Head placement
- Speed control

## Presentation

### Artistic Carriage

**Performance posture:**
- Elegant lines
- Expressive movement
- Strong positions
- Finished movements
- Character portrayal

Good figure skating posture creates the combination of athleticism and artistry the sport demands.
