# Plancha: Ejercicio Fundamental del Core

La plancha es uno de los ejercicios más efectivos para fortalecer el core. Trabaja múltiples músculos simultáneamente en una posición isométrica segura.

## Datos Rápidos

| | |
|---|---|
| **Trabaja** | Core completo, hombros, espalda |
| **Ideal Para** | Fuerza del core, estabilidad, postura |
| **Dificultad** | Principiante a intermedio |
| **Equipo** | Ninguno |
| **Tiempo** | 20-60 segundos |

## Por Qué Funciona

- Activa todos los músculos del core simultáneamente
- Posición segura para la columna
- No requiere movimiento ni equipo
- Fácil de progresar

## Instrucciones

### Posición

1. Comienza en posición de flexión
2. Antebrazos en el suelo (o manos extendidas)
3. Codos directamente bajo los hombros
4. Cuerpo en línea recta de cabeza a talones
5. Core activo, glúteos apretados

### Mantén la Posición

- Respira normalmente
- Mantén la línea recta
- No dejes caer las caderas
- No subas las caderas demasiado
- Mira hacia el suelo

### Tiempo

- **Principiante:** 10-20 segundos
- **Intermedio:** 30-45 segundos
- **Avanzado:** 60+ segundos

## Errores Comunes

- **Caderas caídas** — Activa más el core
- **Caderas muy altas** — Baja ligeramente
- **Cabeza caída** — Mira al suelo, cuello alineado
- **Hombros encogidos** — Empuja el suelo, hombros abajo
- **Contener la respiración** — Respira continuamente

## Variaciones

### Plancha de Rodillas

- Rodillas en el suelo
- Mantén línea recta desde cabeza a rodillas
- Ideal para principiantes

### Plancha Alta

- Sobre las manos extendidas
- Brazos rectos
- Muñecas bajo hombros

### Plancha Lateral

- Apoyada en un antebrazo
- Cuerpo de lado
- Trabaja los oblicuos

## Progresión

1. Empieza con plancha de rodillas
2. Progresa a plancha completa por 20 segundos
3. Aumenta tiempo gradualmente
4. Añade variaciones

## Puntos Clave

1. **Calidad sobre cantidad** — Mejor forma que más tiempo
2. **Línea recta** — De cabeza a talones
3. **Core activo** — Ombligo hacia la columna
4. **Respira** — No contengas el aire
5. **Progresa gradualmente** — No fuerces el tiempo
