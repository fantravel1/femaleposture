# Bicho Muerto (Dead Bug): Estabilidad del Core

El bicho muerto enseña a tu core a mantener la estabilidad mientras las extremidades se mueven. Es seguro para la espalda baja y excelente para la postura.

## Datos Rápidos

| | |
|---|---|
| **Trabaja** | Core profundo, coordinación |
| **Ideal Para** | Estabilidad, dolor de espalda, postura |
| **Dificultad** | Principiante |
| **Equipo** | Colchoneta |
| **Tiempo** | 3-5 minutos |

## Por Qué Funciona

- Entrena el core en su función real (estabilizar)
- Seguro para la columna
- Mejora la coordinación
- Transfiere a movimientos de la vida diaria

## Instrucciones

### Posición Inicial

1. Acuéstate boca arriba
2. Brazos extendidos hacia el techo
3. Caderas y rodillas a 90°
4. Espalda baja presionada ligeramente contra el suelo
5. Core activo

### El Movimiento

1. Exhala y activa el core
2. Extiende el brazo derecho hacia atrás
3. Simultáneamente extiende la pierna izquierda
4. Mantén la espalda baja en contacto con el suelo
5. Regresa con control
6. Repite del otro lado
7. Alterna

### Repeticiones

- 8-12 repeticiones por lado
- 2-3 series
- Enfócate en no arquear la espalda

## Errores Comunes

- **Arquear la espalda baja** — El error más común; reduce el rango
- **Contener la respiración** — Exhala al extender
- **Mover demasiado rápido** — El control es esencial
- **Perder la posición neutral** — Mantén la espalda estable

## Variaciones

### Más Fácil

- Solo extiende las piernas (brazos quietos)
- Solo extiende los brazos (piernas quietas)
- Reduce el rango de extensión

### Más Difícil

- Añade una pausa en la extensión
- Ralentiza el movimiento
- Extiende completamente la pierna hasta el suelo

## Puntos Clave

1. **La espalda baja es la prioridad** — No debe arquearse
2. **Exhala al extender** — Ayuda a mantener el core
3. **Lento y controlado** — No es una carrera
4. **Opuestos juntos** — Brazo derecho con pierna izquierda
5. **Progresa gradualmente** — Aumenta el rango con el tiempo
