# Perro-Pájaro: Ejercicio de Estabilidad del Core

El perro-pájaro es un ejercicio fundamental que fortalece el core, los glúteos y la espalda mientras desafía el equilibrio y la coordinación.

## Datos Rápidos

| | |
|---|---|
| **Trabaja** | Core, glúteos, espalda baja, hombros |
| **Ideal Para** | Estabilidad del core, dolor de espalda, equilibrio |
| **Dificultad** | Principiante |
| **Equipo** | Colchoneta (opcional) |
| **Tiempo** | 3-5 minutos |

## Por Qué Funciona

- Fortalece los músculos estabilizadores profundos
- Mejora la coordinación entre lados del cuerpo
- Entrena el core sin estresar la columna
- Desarrolla el equilibrio y la propiocepción

## Instrucciones

### Posición Inicial

1. Comienza en cuatro puntos
2. Muñecas bajo hombros, rodillas bajo caderas
3. Columna neutral, core ligeramente activo
4. Mira hacia el suelo

### El Movimiento

1. **Extiende el brazo derecho** hacia adelante (pulgar hacia arriba)
2. **Simultáneamente extiende la pierna izquierda** hacia atrás
3. Mantén la pelvis y los hombros nivelados
4. Sostén 3-5 segundos
5. Regresa a la posición inicial con control
6. Repite del otro lado

### Repeticiones

- 8-12 repeticiones por lado
- 2-3 series
- Alterna los lados

## Errores Comunes

- **Arquear la espalda baja** — Mantén el core activo
- **Rotar las caderas** — La pelvis permanece estable
- **Sostener la respiración** — Respira normalmente
- **Ir demasiado rápido** — El control es clave

## Modificaciones

### Más Fácil

- Solo extiende el brazo (sin la pierna)
- Solo extiende la pierna (sin el brazo)
- Reduce el tiempo de sostén

### Más Difícil

- Aumenta el tiempo de sostén a 10 segundos
- Añade pequeños pulsos arriba
- Usa una superficie inestable

## Puntos Clave

1. **Estabilidad primero** — No sacrifiques la forma
2. **Movimiento lento y controlado** — Sin balanceos
3. **Respira normalmente** — No contengas el aire
4. **Pelvis nivelada** — Es el mayor desafío
