# Plancha Lateral: Fortalece el Core Lateral

La plancha lateral es excelente para fortalecer los oblicuos y desarrollar estabilidad lateral.

## Datos Rápidos

| | |
|---|---|
| **Trabaja** | Oblicuos, glúteo medio, hombros |
| **Ideal Para** | Estabilidad lateral, equilibrio, core |
| **Dificultad** | Intermedio |
| **Equipo** | Colchoneta |
| **Tiempo** | 2-3 minutos |

## Instrucciones

1. Acuéstate de lado
2. Apóyate en el antebrazo (codo bajo el hombro)
3. Piernas extendidas, pies apilados
4. Levanta las caderas del suelo
5. Crea una línea recta de cabeza a pies
6. Sostén 15-30 segundos
7. Cambia de lado

## Errores Comunes

- **Caderas caídas** — Mantén la línea recta
- **Rotar hacia adelante** — Pecho abierto
- **Hombro hundido** — Empuja el suelo

## Modificaciones

### Más Fácil

- Rodillas dobladas, apoyada en rodillas
- Reduce el tiempo

### Más Difícil

- Levanta la pierna superior
- Añade movimiento de cadera
- Brazo superior hacia el techo

## Puntos Clave

1. **Línea recta** — De cabeza a pies
2. **Core activo** — No dejes caer las caderas
3. **Hombro estable** — No te hundas
4. **Ambos lados** — Trabaja por igual
