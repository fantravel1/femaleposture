# Pregnancy Posture Checklist

> **Your Body Is Building a Human:** Posture challenges change throughout pregnancy. Use this checklist to stay ahead of the changes and support your changing body.

---

## First Trimester Checklist

### Setup & Preparation

- [ ] Baseline posture assessment (photo/notes)
- [ ] Comfortable supportive shoes identified
- [ ] Pregnancy pillow researched/purchased
- [ ] Healthcare provider consulted about exercise
- [ ] Current exercise routine reviewed for safety

### Daily Practices

- [ ] Morning posture check
- [ ] Pelvic floor exercises (Kegels) started
- [ ] Core engagement practice (while still safe)
- [ ] Regular walking or movement
- [ ] Avoiding high heels for long periods

### Watch For

- [ ] Fatigue affecting posture
- [ ] Nausea affecting movement willingness
- [ ] Breast tenderness requiring bra adjustments
- [ ] Any unusual pain (report to provider)

### Equipment Needed

- [ ] Supportive bra (may need upsizing)
- [ ] Comfortable flat shoes
- [ ] Pregnancy pillow (for later, but buy now)

---

## Second Trimester Checklist

### Adjustments Needed

- [ ] Workspace ergonomics reviewed
- [ ] Sleeping position transitioning to side
- [ ] Car seat position adjusted
- [ ] Bag/purse weight reduced
- [ ] Maternity support belt considered

### Daily Practices

- [ ] Posture reset at least 3x daily
- [ ] Hip stretches daily
- [ ] Back stretches daily
- [ ] Pelvic tilts practiced
- [ ] Walking maintained (or started)
- [ ] Avoiding lying flat on back for exercise

### Body Awareness

- [ ] Center of gravity shifting—wider stance adopted
- [ ] Ligament loosening—avoiding overstretching
- [ ] Baby bump affecting reach—modifying activities
- [ ] Balance changes—moving more carefully

### Watch For

- [ ] Round ligament pain
- [ ] Lower back discomfort
- [ ] Pelvic girdle pain
- [ ] Sciatica symptoms
- [ ] Hip pain at night

### Equipment Needed

- [ ] Maternity support belt (if needed)
- [ ] Body pillow or pregnancy pillow
- [ ] Lumbar support for car/desk
- [ ] Maternity clothing that doesn't restrict

---

## Third Trimester Checklist

### Major Adjustments

- [ ] Sleeping fully side-lying with support
- [ ] Exercise modified significantly
- [ ] Work accommodations requested if needed
- [ ] Birth preparation including positions practice
- [ ] Postpartum setup planned

### Daily Practices

- [ ] Frequent position changes (every 30 min)
- [ ] Gentle stretching only (no deep stretches)
- [ ] Pelvic floor exercises continued
- [ ] Breathing exercises for rib expansion
- [ ] Rest periods increased
- [ ] Walking or gentle movement daily

### Body Management

- [ ] Accepting wider stance is necessary
- [ ] Using support when getting up
- [ ] Avoiding twisting with movement
- [ ] Rolling to side before sitting up
- [ ] Asking for help with reaching/bending

### Watch For

- [ ] Significant increase in back pain
- [ ] Pubic symphysis pain
- [ ] Carpal tunnel symptoms
- [ ] Swelling in feet/legs
- [ ] Balance issues
- [ ] Breathing difficulties

### Equipment/Setup

- [ ] Multiple pillows for sleep positioning
- [ ] Stool for feet when sitting
- [ ] Support for getting out of bed
- [ ] Hospital bag packed with comfortable clothes

---

## Sitting Posture Checklist (All Trimesters)

### Setup

- [ ] Chair supports lower back
- [ ] Feet flat on floor or footrest
- [ ] Thighs parallel to floor
- [ ] Screen at eye level
- [ ] Keyboard at elbow height

### During Sitting

- [ ] Avoid crossing legs (especially later)
- [ ] Stand up every 30 minutes
- [ ] Shift position regularly
- [ ] Keep water nearby (stay hydrated)
- [ ] Avoid slouching into chair

---

## Sleeping Posture Checklist (All Trimesters)

### First Trimester

- [ ] Any comfortable position still okay
- [ ] Start practicing side-lying
- [ ] Consider pregnancy pillow early

### Second Trimester

- [ ] Transitioning to side sleeping
- [ ] Left side preferred (better circulation)
- [ ] Pillow between knees
- [ ] Support for belly as grows

### Third Trimester

- [ ] Full side-lying position
- [ ] Multiple pillow support system
- [ ] No flat-on-back sleeping
- [ ] Support to reduce hip pain
- [ ] Easy exit strategy from bed

---

## Exercise Posture Checklist

### Safe Movement Principles

- [ ] Avoid supine exercise after first trimester
- [ ] No deep stretching (ligaments are loose)
- [ ] Modify for growing belly
- [ ] Stay well-hydrated
- [ ] Stop if anything feels wrong
- [ ] Talk test for intensity

### Exercises to Continue (Modified)

- [ ] Walking
- [ ] Swimming
- [ ] Prenatal yoga
- [ ] Modified strength training
- [ ] Pelvic floor exercises

### Exercises to Avoid

- [ ] Contact sports
- [ ] High fall-risk activities
- [ ] Heavy lifting
- [ ] Hot yoga/hot tub
- [ ] Exercises lying flat (after T1)
- [ ] Deep core work that causes coning

---

## When to Get Help

### See Provider If

- [ ] Pain interferes with daily activities
- [ ] Symptoms are worsening despite modifications
- [ ] New numbness or tingling
- [ ] Pelvic floor symptoms (leaking, pressure)
- [ ] Unable to sleep due to discomfort
- [ ] Any concerning symptoms

### Specialists Who Can Help

- [ ] Physical therapist (pelvic health specialty)
- [ ] Prenatal exercise specialist
- [ ] Massage therapist (prenatal certified)
- [ ] Chiropractor (prenatal experience)

---

## Postpartum Preparation

### Before Baby Arrives

- [ ] Nursing pillow purchased
- [ ] Bassinet/changing station at good height
- [ ] Recovery support plan
- [ ] Postpartum exercise plan discussed with provider
- [ ] Pelvic floor PT appointment scheduled

### Know About Recovery

- [ ] Posture challenges continue postpartum
- [ ] Core and pelvic floor need rehab
- [ ] Feeding positions affect posture
- [ ] Baby carrying requires attention
- [ ] Full recovery takes months
