# Home Office Setup Checklist

> **Your Home Is Now Your Workplace:** Unlike offices, home setups are rarely ergonomically designed. This checklist helps you create a workspace that protects your posture.

---

## Quick Assessment

### Current Problem Signs

Check if you experience:
- [ ] Neck pain after work
- [ ] Lower back discomfort
- [ ] Shoulder/upper back tension
- [ ] Headaches in afternoon
- [ ] Wrist or hand pain
- [ ] Eye strain
- [ ] Fatigue by mid-day

If you checked any, your setup needs work.

---

## The Chair

### Essentials

- [ ] **Height adjustable** to allow feet flat on floor
- [ ] **Seat depth** allows 2-3 fingers between seat edge and back of knees
- [ ] **Back support** that matches your spine curve
- [ ] **Armrests** at height where shoulders stay relaxed

### Optimal Settings

- [ ] Feet flat on floor (or on footrest)
- [ ] Thighs parallel to floor
- [ ] Hips at 90-100 degrees
- [ ] Back fully supported
- [ ] Lumbar curve supported
- [ ] Armrests supporting forearms at desk height

### If Your Chair Doesn't Adjust

**Add-on solutions:**
- [ ] Seat cushion to adjust height
- [ ] Lumbar roll or pillow for back support
- [ ] Footrest if feet don't reach floor
- [ ] Arm supports that attach to desk

### Red Flags

- Chair is too soft/unsupportive
- Sitting on couch or bed to work
- Bar stool or dining chair without support
- Chair height cannot be adjusted

## The Desk

### Height

- [ ] **Seated:** Elbows at 90 degrees when typing
- [ ] **Standing:** Sameâ€”elbows at 90 degrees
- [ ] Wrists neutral, not bent up or down
- [ ] Shoulders relaxed, not shrugged

### Standard Heights

- Seated desk: 28-30 inches for most people
- May need adjustment based on your height
- Keyboard tray can help if desk is too high

### Surface Space

- [ ] Enough room for monitor at proper distance
- [ ] Space for keyboard and mouse at same level
- [ ] Room to keep frequently used items within reach
- [ ] Not cluttered with non-essential items

### Adjustable Desk Options

- [ ] Standing desk (full electric adjustment)
- [ ] Desk converter (sits on existing desk)
- [ ] Keyboard tray (lowers keyboard position)

## The Monitor

### Height

- [ ] Top of screen at or slightly below eye level
- [ ] You're looking slightly down at center of screen
- [ ] Not tilting head up or down significantly

### Distance

- [ ] About arm's length away (20-26 inches)
- [ ] Text readable without leaning forward
- [ ] Full screen visible without excessive eye movement

### Angle

- [ ] Tilted slightly back (10-20 degrees)
- [ ] Directly in front of you (not to the side)
- [ ] No glare from windows or lights

### Solutions for Problems

**Too low:**
- [ ] Monitor arm/stand
- [ ] Stack of books (temporary)
- [ ] Laptop stand (for laptops)

**Too far/close:**
- [ ] Adjust desk depth
- [ ] Monitor arm for adjustability

**Dual monitors:**
- [ ] Primary monitor directly in front
- [ ] Secondary at slight angle
- [ ] Both at same height

### Laptop Considerations

- [ ] Laptop on stand at eye level
- [ ] External keyboard and mouse
- [ ] Never work on laptop in lap for extended periods

---

## Keyboard and Mouse

### Keyboard Position

- [ ] At elbow height
- [ ] Wrists neutral (not bent up or down)
- [ ] Shoulders relaxed
- [ ] Elbows at sides (not reaching)

### Keyboard Features to Consider

- [ ] Compact/tenkeyless to keep mouse closer
- [ ] Slight negative tilt (back higher than front)
- [ ] Wrist rest for breaks (not while typing)
- [ ] Split/ergonomic keyboard if wrist issues

### Mouse Position

- [ ] Same level as keyboard
- [ ] As close to keyboard as possible
- [ ] Arm moves from shoulder, not wrist
- [ ] Mouse fits your hand size

### Mouse Alternatives

- [ ] Trackpad
- [ ] Vertical mouse (reduces forearm rotation)
- [ ] Trackball
- [ ] Switch between options

---

## Lighting

### General Lighting

- [ ] Even, bright enough to see without strain
- [ ] No harsh shadows on work surface
- [ ] Not too bright (causes squinting)

### Screen Lighting

- [ ] No glare on monitor
- [ ] Not facing window directly
- [ ] Not window directly behind you (backlight)
- [ ] Bias lighting behind monitor (optional, reduces strain)

### Task Lighting

- [ ] Adjustable lamp for documents if needed
- [ ] Light on documents similar to screen brightness
- [ ] Position to avoid reflections

---

## The Work Environment

### Temperature

- [ ] Comfortable (not too cold, which causes tensing)
- [ ] Able to adjust layers
- [ ] Not shivering or sweating

### Noise

- [ ] Manageable background noise
- [ ] Noise-canceling headphones if needed
- [ ] Door that closes if possible

### Phone/Tablet Setup

- [ ] Phone stand to avoid looking down
- [ ] Headset/speakerphone for calls (no cradling phone)
- [ ] Tablet at eye level if used for work

---

## Movement Reminders

### Equipment

- [ ] Timer or app for break reminders
- [ ] Standing desk or standing option
- [ ] Space for stretching nearby
- [ ] Walking path for movement breaks

### Schedule

- [ ] Stand every 30-60 minutes
- [ ] Move/stretch every hour
- [ ] Take lunch away from desk
- [ ] Walk meetings when possible

## Room-by-Room Assessment

### The Ideal Home Office

- Dedicated room with door
- Desk at correct height
- Quality chair
- Good lighting
- Temperature control
- Minimal distractions

### Kitchen Table Setup

**Challenges:** Chair height, desk height, lighting
**Solutions:**
- Chair cushion for height
- Laptop stand
- External keyboard/mouse
- Good overhead lighting

### Couch/Living Room

**Challenges:** No back support, laptop on lap, looking down
**Solutions:**
- Lap desk with slope
- Back support cushion
- Limit to 30-minute sessions
- Move to proper desk for focused work

### Bedroom

**Challenges:** Bad for sleep hygiene, often poor setup
**Solutions:**
- If must use, set up at dresser height
- Stand while working if no desk
- Remove work setup at end of day if possible

## Testing Your Setup

### The 15-Minute Test

1. Set up according to this checklist
2. Work for 15 minutes
3. Note any discomfort
4. Adjust and test again
5. Repeat until comfortable

### Daily Check

Before starting work:
- [ ] Chair at right height
- [ ] Lumbar support in place
- [ ] Monitor at eye level
- [ ] Feet flat on floor
- [ ] Shoulders relaxed

### Weekly Review

- [ ] Any new aches or pains?
- [ ] Taking regular breaks?
- [ ] Equipment still adjusted correctly?
- [ ] What needs adjustment?
